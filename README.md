# Elysia with Bun runtime

## Getting Started

To get started with this template, simply paste this command into your terminal:

```bash
bun create elysia .
```

## Development

To start the development server run:

```bash
bun run dev
```

## Routes

| method | path      |
| ------ | --------- |
| get    | /         |
| get    | /users    |
| post   | /user     |
| get    | /user/:id |
| put    | /user/:id |
| delete | /user/:id |
